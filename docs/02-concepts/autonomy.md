# 概念：自主性

## 当前定义

自主性是智能体在没有逐步人工指令的情况下，根据目标选择行动、调用工具并推进任务的能力。

## 自主性不是无限行动权

更高的自主性必须配套更强的边界、权限和可观察性。

自主性增长应同时回答：

- 智能体可以自己决定什么？
- 哪些动作必须经过用户确认？
- 如何记录智能体做过什么？
- 如何中断或回滚？
- 如何判断它没有偏离目标？

## 自主性与可控性

```text
低自主性：用户逐步指挥，安全但低效
高自主性：智能体连续行动，高效但风险更高
理想状态：在清晰边界内自主行动，并让人类随时理解和接管
```

## 初始设计倾向

第一阶段采用低到中等自主性。智能体可以读取和修改工作区文件、运行安全命令，但涉及破坏性操作、外部网络、系统级修改时需要显式确认。
