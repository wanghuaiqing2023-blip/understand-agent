# 概念：智能体

## 当前定义

智能体不是单纯的聊天机器人，而是能够围绕目标进行任务理解、计划生成、工具调用、执行反馈和结果总结的任务执行系统。

## 与聊天机器人的区别

聊天机器人主要生成文本回应。

智能体需要在真实环境中产生行动：

- 读取信息
- 操作工具
- 修改文件
- 执行命令
- 验证结果
- 根据反馈继续调整

## 最小结构

```text
用户目标
  -> 任务理解
  -> 计划生成
  -> 工具选择
  -> 行动执行
  -> 结果观察
  -> 迭代修正
  -> 最终总结
```

## 关键问题

- 智能体如何理解任务边界？
- 智能体如何知道应该调用哪个工具？
- 智能体如何判断任务已经完成？
- 智能体如何处理失败？
- 智能体如何让人类保持掌控？
